# 🎓 EduGenie – AI-Powered Educational Assistant

EduGenie is a lightweight AI-powered educational assistant designed to simplify and enhance the learning experience using the power of Generative AI. Built for students, self-learners, and educators, it provides intelligent educational support through question answering, concept explanation, quiz generation, text summarization, and personalized learning recommendations.

Developed with **FastAPI** for the backend and a responsive **HTML, CSS, and Jinja2** frontend, EduGenie integrates the capabilities of **Google Gemini 1.5 Pro** and **LaMini-Flan-T5** to deliver fast, accurate, and context-aware educational assistance. Its lightweight architecture enables efficient execution even on resource-constrained devices such as the Apple Mac M1, making it suitable for personal learning, academic projects, and AI education applications.

## 🎯 Use Cases

### Intelligent Question Answering

Students can ask educational questions such as *"Which is the largest ocean?"* and receive accurate answers along with additional learning context.

### Quiz Generation

Learners can generate topic-specific quizzes to assess their understanding. For example, selecting **Generate Quiz** on the *Pythagoras Theorem* produces multiple assessment questions instantly.

### Personalized Learning Roadmaps

Users interested in topics like **SQL**, **Machine Learning**, or **Data Structures** can receive structured learning paths covering beginner, intermediate, and advanced concepts with recommended progression.

### Educational Summarization

Lengthy educational content can be summarized into concise, easy-to-understand notes for faster revision.
